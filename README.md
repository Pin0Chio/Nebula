# Nebula

> The Roaming Kit

## About

Nebula is an open-source investigation toolkit.

The goal of Nebula is to help explore, analyze, and understand technical artifacts such as:

- SQLite databases
- Binary property lists (bplists)
- Crash logs
- System logs
- File structures
- Other forensic and diagnostic artifacts

Nebula is being built one investigation at a time.

## Philosophy

- Evidence over assumptions.
- Read-only by default.
- Learn by exploring.
- Share knowledge with others.
